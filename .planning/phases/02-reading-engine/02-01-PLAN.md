**ORPWord.swift** -- struct with:
- `text: String` (the full word)
- `orpIndex: Int` (zero-based character index of the ORP letter)
- `beforeORP: String` (characters before ORP letter)
- `orpCharacter: String` (the single ORP letter)
- `afterORP: String` (characters after ORP letter)
- `isSentenceEnd: Bool` (from WordToken)
- `wordIndex: Int` (global index in chapter)
- Static factory `from(token: WordToken)` that calculates ORP using lookup table:
  - Word length 1-2: position 0, 3-6: position 1, 7-10: position 2, 11-13: position 3, 14+: position 4
  - Source: speedread open-source implementation (verified in research)
- Split text into beforeORP/orpCharacter/afterORP based on calculated orpIndex

**VoiceInfo.swift** -- struct with:
- `identifier: String` (AVSpeechSynthesisVoice identifier)
- `name: String` (display name)
- `language: String`
- `quality: Quality` enum (default, enhanced, premium)
- `isInstalled: Bool`
- Static factory `from(voice: AVSpeechSynthesisVoice)` that maps quality tier and determines installed status (enhanced/premium only appear if installed)

**RSVPEngine.swift** -- @Observable class with:
- Published state: `currentWord: ORPWord?`, `isPlaying: Bool`, `currentIndex: Int`, `wordCount: Int`
- `@ObservationIgnored` private state: words array `[ORPWord]`, Timer reference, wpm setting
- `loadChapter(text: String)` -- uses existing WordTokenizer to tokenize, converts each WordToken to ORPWord, stores array, resets index to 0
- `play()` -- starts Timer.scheduledTimer that calls `advanceToNextWord()` at interval derived from WPM
- `pause()` -- invalidates timer, sets isPlaying = false. Last word stays on currentWord (per CONTEXT.md locked decision: frozen display on pause)
- `resume()` -- backs up 3-5 words (use 4 as default, per Claude's discretion) before pause point, then starts timer. Clamp to 0 minimum.
- `setWPM(_ wpm: Int)` -- updates WPM (clamped 100-500), recalculates timer interval if playing. WPM drives exact timing when TTS is off.
- `seekTo(index: Int)` -- sets currentIndex, updates currentWord
- `word(at index: Int) -> ORPWord?` -- returns word at index for coordinator use
- Private `advanceToNextWord()` -- increments index, sets currentWord, calculates next delay using punctuation-aware timing:
  - Base interval = 60.0 / WPM
  - Length penalty: +0.04 * sqrt(wordLength) added to multiplier
  - Sentence-ending (.!?): multiplier 3.0x
  - Clause punctuation (,;:): multiplier 2.0x
  - Standard word: multiplier 0.9x (slightly less than full interval per speedread)
  - Reschedule timer with calculated delay for next word
- Private `scheduleNextWord(delay: TimeInterval)` -- invalidates current timer, schedules new one-shot timer for the delay, fires advanceToNextWord on main RunLoop
- When reaching end of words array: set isPlaying = false, post a notification or call a closure `onChapterComplete: (() -> Void)?` for coordinator to handle chapter advancement

**TTSService.swift** -- @Observable class following the NSObject delegate bridge pattern from research:

**Observable state:**
- `isSpeaking: Bool`
- `currentGlobalWordIndex: Int` (word index in the full chapter, across all sentences)

**@ObservationIgnored private state:**
- `synthesizer: AVSpeechSynthesizer?` (optional -- recreated per chapter for iOS 17+ reliability)
- `delegateHandler: DelegateHandler!` (inner NSObject, see below)
- `sentenceQueue: [(text: String, wordOffset: Int)]` -- prepared sentences with cumulative word offsets
- `currentSentenceIndex: Int`
- `selectedVoiceIdentifier: String?` -- voice to use for speech
- Callback: `onWordBoundary: ((Int) -> Void)?` -- fires with global word index for coordinator
- Callback: `onChapterComplete: (() -> Void)?` -- fires when all sentences spoken

**Inner class DelegateHandler: NSObject, AVSpeechSynthesizerDelegate:**
- `weak var owner: TTSService?`
- `speechSynthesizer(_:willSpeakRangeOfSpeechString:utterance:)` -- convert character range to word index within the sentence text, add sentenceQueue[currentSentenceIndex].wordOffset to get global word index. Dispatch to MainActor: update owner's currentGlobalWordIndex and call onWordBoundary. Word index calculation: count spaces before the range.location in utterance.speechString to get sentence-local word index.
- `speechSynthesizer(_:didFinish:)` -- on MainActor: increment currentSentenceIndex. If more sentences remain, call speakNextSentence(). If no more sentences, set isSpeaking = false and call onChapterComplete.
- `speechSynthesizer(_:didCancel:)` -- set isSpeaking = false on MainActor.

**Public methods:**
- `prepareChapter(_ text: String)` -- use NLTokenizer(unit: .sentence) pinned to .english (matching WordTokenizer convention). Enumerate sentences, build sentenceQueue with cumulative word offsets. Word count per sentence: use NLTokenizer(unit: .word) for consistency with WordTokenizer (not String.split). Reset currentSentenceIndex to 0.
- `speak(fromWordIndex: Int)` -- find which sentence contains fromWordIndex (binary search or linear scan through wordOffsets), set currentSentenceIndex, create fresh AVSpeechSynthesizer instance (per anti-pattern: recreate, don't reuse), set synthesizer.usesApplicationAudioSession = false (per WWDC 2020), attach delegateHandler, call speakNextSentence() starting mid-sentence if needed. Set isSpeaking = true.
- `pause()` -- synthesizer?.pauseSpeaking(at: .immediate). Keep synthesizer alive (pause is safe, only stop is problematic).
- `resume()` -- synthesizer?.continueSpeaking()
- `stop()` -- synthesizer?.stopSpeaking(at: .immediate), then nil out synthesizer (per anti-pattern: recreate after stop). Set isSpeaking = false.
- `setVoice(identifier: String)` -- store voice identifier for next utterance creation
- `setRate(_ rate: Float)` -- store rate (0.0-1.0) for next utterance creation

**Private methods:**
- `speakNextSentence()` -- guard currentSentenceIndex < sentenceQueue.count. Create AVSpeechUtterance with sentenceQueue[currentSentenceIndex].text. Set voice from selectedVoiceIdentifier (AVSpeechSynthesisVoice(identifier:)). Set rate. Set preUtteranceDelay = 0, postUtteranceDelay = 0.05 (tiny gap between sentences for natural flow). synthesizer?.speak(utterance). Only queue ONE sentence at a time (per anti-pattern: don't queue many).
- `wordIndexFromCharRange(_ range: NSRange, in text: String)` -- count NLTokenizer word tokens before the range start position in the sentence text.

**Thread safety:** All delegate callbacks dispatch to MainActor via `Task { @MainActor in ... }` before updating @Observable properties (per Pitfall 4).
