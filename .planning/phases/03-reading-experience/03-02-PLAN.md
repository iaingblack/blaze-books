    **PageModeView.swift:**
    Create a SwiftUI view that renders chapter text as highlighted paragraphs with auto-scrolling:

    1. **Parameters:**
       - `paragraphs: [ParagraphData]` — from PageTextService.splitIntoParagraphs
       - `highlightedWordIndex: Int?` — from ReadingCoordinator.highlightedWordIndex
       - `chapterTitle: String` — chapter heading
       - `pageTextService: PageTextService` — for attributedString generation

    2. **State:**
       - `@State private var lastScrolledParagraph: Int = -1` — tracks last auto-scrolled paragraph to avoid redundant scrolls

    3. **Body structure:**
       ```
       ScrollViewReader { proxy in
           ScrollView {
               LazyVStack(alignment: .leading, spacing: 0) {
                   // Chapter header (same style as existing ReadingView)
                   Text(chapterTitle).font(.title).fontWeight(.bold)
                       .padding(.horizontal, 20).padding(.top, 24).padding(.bottom, 16)
                       .id("chapter-header")

                   // Paragraphs with word highlighting
                   ForEach(paragraphs) { paragraph in
                       Text(pageTextService.attributedString(for: paragraph, highlightedWordIndex: highlightedWordIndex))
                           .font(.system(size: 17))
                           .lineSpacing(7)
                           .textSelection(.enabled)
                           .padding(.horizontal, 20)
                           .padding(.bottom, 14)
                           .id(paragraph.id)
                           .transaction { $0.animation = nil }  // No animation on highlight swap (Research anti-pattern)
                   }

                   Spacer().frame(height: 100)
               }
           }
           .onChange(of: highlightedWordIndex) { _, newIndex in
               // Auto-scroll to paragraph containing highlighted word
               guard let newIndex = newIndex else { return }
               if let pIdx = pageTextService.paragraphIndex(forWordIndex: newIndex, paragraphs: paragraphs),
                  pIdx != lastScrolledParagraph {
                   lastScrolledParagraph = pIdx
                   withAnimation(.easeInOut(duration: 0.3)) {
                       proxy.scrollTo(paragraphs[pIdx].id, anchor: .center)
                   }
               }
           }
       }
       ```

    4. **Scroll position on appear:**
       - If highlightedWordIndex is non-nil on appear (e.g., switching from RSVP while paused), scroll to the paragraph containing that word immediately
       - Use `.onAppear { if let idx = highlightedWordIndex { ... proxy.scrollTo } }`

    5. **Broken chapter handling:**
       - If paragraphs is empty, show the same broken chapter placeholder as existing ReadingView (Image systemName exclamationmark.triangle + message)

    **WPMSliderView.swift:**
    Extract the WPM slider into a reusable component shared by both modes:

    1. **Parameters:**
       - `@Binding var sliderWPM: Double` — bound to parent's local slider state
       - `effectiveWPM: Int` — from coordinator, shown as display value
       - `isSpeedCapped: Bool` — from coordinator, for visual feedback
       - `onWPMChanged: (Int) -> Void` — called continuously during drag (updates RSVPEngine immediately)
       - `onWPMChangeEnded: (Int) -> Void` — called when drag ends (triggers TTS restart)

    2. **Body:**
       ```
       VStack(spacing: 4) {
           HStack {
               Text("100").font(.caption2).foregroundStyle(.secondary)
               Slider(value: $sliderWPM, in: 100...500, step: 10) {
                   Text("WPM")
               } onEditingChanged: { editing in
                   if !editing {
                       onWPMChangeEnded(Int(sliderWPM))
                       sliderWPM = Double(effectiveWPM)  // Snap to effective after cap
                   }
               }
               .onChange(of: sliderWPM) { _, newValue in
                   onWPMChanged(Int(newValue))
               }
               Text("500").font(.caption2).foregroundStyle(.secondary)
           }
           .padding(.horizontal, 20)
           .padding(.vertical, 8)
           .background(.bar)
       }
       ```

    This extracts the existing inline slider from ReadingView into a reusable component.
