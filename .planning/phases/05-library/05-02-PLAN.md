Create `BlazeBooks/Services/LibraryService.swift`:
- A struct (stateless service, like WordTokenizer pattern) with static methods
- `static func deleteBook(_ book: Book, modelContext: ModelContext)`:
  1. Capture `book.filePath` before deletion
  2. Call `modelContext.delete(book)` -- SwiftData cascade rules handle chapter and reading position cleanup, and `.nullify` on shelves automatically removes book from shelf.books arrays
  3. Delete the EPUB file via `try FileStorageManager.deleteFile(filePath)` wrapped in do/catch -- log warning on failure but don't crash (orphaned file is less harmful than orphaned record, per research Pitfall 3)
- `static func addBookToShelf(_ book: Book, _ shelf: Shelf)`:
  - Guard that shelf.books doesn't already contain this book (check by id)
  - Append book to `shelf.books`
- `static func removeBookFromShelf(_ book: Book, _ shelf: Shelf)`:
  - Remove book from `shelf.books` by id
- `static func createShelf(name: String, modelContext: ModelContext) -> Shelf`:
  - Create new Shelf with the given name
  - Insert into modelContext BEFORE returning (per research Pitfall 2: insert before relating)
  - Return the shelf
- `static func deleteShelf(_ shelf: Shelf, modelContext: ModelContext)`:
  - Call `modelContext.delete(shelf)` -- `.nullify` ensures books are NOT deleted
- `static func renameShelf(_ shelf: Shelf, to newName: String)`:
  - Set `shelf.name = newName`

Create `BlazeBooks/Views/Library/ShelfSectionView.swift`:
- Takes a `Shelf` binding or observed shelf, plus an `isExpanded` Binding<Bool>
- Also takes a `shelves: [Shelf]` array (for the context menu's "Add to Shelf" submenu on books within the section)
- Uses `DisclosureGroup(isExpanded: isExpanded)` with:
  - Label: shelf name in `.font(.title3).bold()` with a context menu for shelf management:
    - "Rename" button that triggers a rename alert (pass via callback or @State)
    - "Delete Shelf" button with `.destructive` role
  - Content: `LazyVGrid` with adaptive columns (120pt min, matching All Books grid) showing `BookCoverView` for each book in `shelf.books ?? []`, each wrapped in `NavigationLink(value: book)`
- If `shelf.books` is empty or nil, show a subtle "No books" placeholder text inside the section
