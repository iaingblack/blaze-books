    **EPUBImportService refactoring (BlazeBooks/Services/EPUBImportService.swift):**
    Extract the core import logic (steps 5-10 from the existing importEPUB method: parse with Readium, create Book/Chapter/ReadingPosition records, insert into modelContext) into a new shared method:

    ```
    func importLocalEPUB(at localURL: URL, modelContext: ModelContext, gutenbergId: Int? = nil) async throws
    ```

    This method:
    1. Compute file hash via FileStorageManager.computeFileHash(at:)
    2. Check for duplicate by file hash (FetchDescriptor query)
    3. If duplicate found, throw a specific error (e.g., ImportError.alreadyInLibrary)
    4. Parse EPUB with Readium (parserService.parseEPUB)
    5. Determine title/author with smart fallbacks (use localURL filename for fallback title)
    6. Create Book record with coverData, fileHash, and optional gutenbergId
    7. Create Chapter records
    8. Create ReadingPosition at chapter 0, word 0
    9. Insert into modelContext

    Then refactor the existing `importEPUB(from:modelContext:)` to:
    1. Handle security-scoped access (steps 1-3: access, hash check, copy to sandbox)
    2. Call `try await importLocalEPUB(at: localURL, modelContext: modelContext)` for the core logic
    3. Catch ImportError.alreadyInLibrary and set importError = "Already in your library"
    4. On Readium parse failure, delete the copied file and set appropriate error

    Add an `enum ImportError: Error` with cases: `alreadyInLibrary`, `parseFailed(String)`.

    **BookDownloadService (BlazeBooks/Services/BookDownloadService.swift):**
    Create as @MainActor @Observable final class (matching project service patterns).

    **Types:**
    ```
    enum DownloadState {
        case downloading
        case importing
        case completed
        case failed(String)
    }
    ```

    **Observable state:**
    - `var activeDownloads: [Int: DownloadState] = [:]` (keyed by Gutenberg book ID)

    **Dependencies:**
    - Private let importService: EPUBImportService (injected or created)

    **Public method:**
    ```
    func downloadBook(_ gutendexBook: GutendexBook, modelContext: ModelContext) async
    ```

    Flow:
    1. Guard epubURL is non-nil, else set .failed("No EPUB available")
    2. Set activeDownloads[id] = .downloading
    3. Download EPUB via `URLSession.shared.download(from: epubURL)` -- per research Pitfall 4, the temp file must be moved immediately
    4. Move temp file to sandbox: FileStorageManager.booksDirectory/gutenberg-{id}.epub
    5. If file already exists at destination, remove it first (re-download scenario)
    6. Set activeDownloads[id] = .importing
    7. Call `try await importService.importLocalEPUB(at: localURL, modelContext: modelContext, gutenbergId: gutendexBook.id)`
    8. Set activeDownloads[id] = .completed
    9. On ImportError.alreadyInLibrary, set .completed (book is already there)
    10. On any other error, clean up the downloaded file and set .failed("Download failed. Tap to retry.")

    **Design notes:**
    - Per locked decision: downloaded EPUBs go through the existing EPUB import pipeline
    - Per locked decision: no "Read Now" after download, just state change to .completed
    - Per research Pitfall 4: move temp file immediately after download completes
    - Per research Pitfall 5: duplicate detection handled by importLocalEPUB's file hash check
    - Per research Pitfall 6: author name formatting handled by GutendexBook.primaryAuthor (but the Readium parser will extract the real author from the EPUB metadata, which takes priority)
