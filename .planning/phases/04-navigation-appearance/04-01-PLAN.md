**1. Create `TableOfContentsView.swift`** in `BlazeBooks/Views/Reading/`:

A SwiftUI view presented as a sheet showing the book's chapter list with current chapter indicator.

- Properties:
  - `let chapters: [Chapter]` -- the book's chapters (will be sorted internally)
  - `let currentChapterIndex: Int` -- to highlight the current chapter
  - `let onChapterSelected: (Int) -> Void` -- callback with the chapter index

- Body: `NavigationStack` wrapping a `List` of chapters sorted by `index`.
  - Each row is a `Button` calling `onChapterSelected(chapter.index)`.
  - Row label: `HStack` with `Text(chapter.title)` styled `.foregroundStyle(.primary)`, a `Spacer()`, and a `bookmark.fill` SF Symbol in `.accentColor` when `chapter.index == currentChapterIndex`.
  - `.navigationTitle("Table of Contents")` with `.navigationBarTitleDisplayMode(.inline)`.
  - Add a dismiss button in the trailing toolbar: `ToolbarItem(placement: .confirmationAction)` with a `Button("Done")` that calls `dismiss` via `@Environment(\.dismiss)`.

**2. Modify `ReadingView.swift`:**

a) Add `@State private var showTableOfContents = false` state property.

b) **Extract `jumpToChapter(_ newIndex: Int)` method** from the existing `navigateChapter(direction:)` logic. The new method:
  - Guards `newIndex >= 0 && newIndex < totalChapters`
  - Calls `coordinator.stop()`
  - Sets `currentChapterIndex = newIndex`
  - Calls `loadChapter(at: newIndex)`
  - Reloads coordinator: `coordinator.loadBook(chapterTexts: sortedChapters.map(\.text), startChapter: newIndex, startWord: 0)`
  - Saves position via `positionService.savePosition(...)`

c) **Simplify `navigateChapter(direction:)`** to just call `jumpToChapter(currentChapterIndex + direction)`.

d) **Add TOC button to toolbar.** Add a `ToolbarItem(placement: .topBarLeading)` with a `Button` showing `Image(systemName: "list.bullet")` that sets `showTableOfContents = true`.

e) **Add `.sheet(isPresented: $showTableOfContents)`** presenting `TableOfContentsView`:
  - Pass `chapters: sortedChapters`, `currentChapterIndex: currentChapterIndex`
  - `onChapterSelected`: dismiss sheet (`showTableOfContents = false`) then call `jumpToChapter(chapterIndex)`
  - Add `.presentationDetents([.medium, .large])` for flexible sheet height.

Note: The existing mode toggle is at `.principal` placement. The TOC button goes at `.topBarLeading` -- these do not conflict.
